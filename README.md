<h1>Projeto: API para uma aplicação de um Petshop</h1>

<h2>📌 Sobre o Projeto</h2>
<p>Projeto Back-End para atendimentos realizados para um sistema de PetShop.</p>

<h2>🚀 Tecnologias Utilizadas</h2>
<ul>
  <li><b>JavaScript</b> - vanilla</li>
</ul>

<h2>✨ Funcionalidades</h2>
<ul>
  <li>Adicionar, editar, ler e deletar informações de atendimento e pets no banco de dados MySQL</li>
  <li>Conectar em um banco de dados relacional</li>
  <li>Tratar dados antes de executar alguma ação</li>
  <li>Tratar dados antes de armazenar</li>
  <li>Tratar dados antes de enviar ao front-end</li>
</ul>

<h2>⚙️ Detalhes Técnicos</h2>
<ul>
  <li>API no padrão REST</li>
  <li>biblioteca express utilizada para criar a API</li>
  <li>Banco de dados relacional MySQL utilizado</li>
  <li>ferramentas nodemon, consign, moment e body-parser utilizadas</li>
</ul>
